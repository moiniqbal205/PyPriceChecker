# PyPriceChecker
Product price checker x web scraping demo using Python

- Start project by running the following:
$ pip install requests bs4

- Search Extended HELO for a better understanding of mail transfer protocol and how it was used in this app

- It is recommended that Google 2-step verification be used for this project as enabling it will provide 
access to Google app passwords, which can be hardcoded into the app instead of using the user's actual password
